# Project Vision

## Working Title

**Mycelium Learning Lab**

## Core Idea

This project explores how mushroom networks and mycelial systems can inspire new ways to think about computing education, learning tools, and human-centered technology.

Rather than treating computing as cold, abstract, or inaccessible, this project asks whether natural systems can make computer science feel more alive, creative, and approachable.

## Why This Matters

Many learners encounter computer science as syntax, rules, errors, and rigid logic before they experience it as creativity, systems-thinking, or problem-solving. Mycelium offers a rich metaphor for connection, growth, communication, resource sharing, adaptation, and hidden structure.

That metaphor could help learners understand concepts like:

- networks
- distributed systems
- data flow
- recursion and growth
- dependency graphs
- feedback loops
- resilience
- collaborative systems

## Early Research Questions

1. How can mycelial networks be used as a metaphor for teaching computing concepts?
2. Can ecological metaphors make CS more approachable for young learners or underrepresented students?
3. What kinds of interfaces help learners see code as a living system rather than a static set of commands?
4. How can a learning tool balance playfulness with technical accuracy?
5. What can natural networks teach us about accessibility, collaboration, and human-centered design?

## Possible Outputs

- A research paper or literature review
- A portfolio project
- A small web-based visualization
- A curriculum prototype
- A child-friendly learning tool
- A concept map connecting biology, computing, and education
