# Concept Map

This is a working map of ideas connecting mycelium, computing, and education.

## Mycelium Concepts

- Growth
- Networks
- Hidden structures
- Resource sharing
- Communication
- Adaptation
- Decay and renewal
- Symbiosis
- Resilience

## Computing Concepts

- Graphs
- Nodes and edges
- Distributed systems
- Data flow
- Network protocols
- Dependency trees
- Recursion
- State changes
- Feedback loops
- Fault tolerance

## Learning Concepts

- Visual learning
- Playful exploration
- Scaffolding
- Conceptual bridges
- Beginner-friendly metaphors
- Accessibility
- Human-centered design
- Creative computing

## Possible Connections

| Mycelium Idea | Computing Concept | Learning Use |
|---|---|---|
| Hyphal growth | Graph traversal | Show how systems expand over time |
| Resource sharing | Data transfer | Teach movement of information |
| Network resilience | Fault tolerance | Show how systems adapt when parts fail |
| Symbiosis | System interaction | Teach collaboration between components |
| Hidden underground structure | Backend systems | Explain unseen infrastructure |
| Fruiting bodies | User-facing output | Connect visible results to hidden processes |
