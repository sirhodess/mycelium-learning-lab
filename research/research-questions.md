# Research Questions

## Main Research Question

How can mycelium-inspired network models help make computer science education more creative, accessible, and human-centered?

## Supporting Questions

### Computing Education

- What barriers make text-based programming feel intimidating to new learners?
- How can natural metaphors help learners understand abstract computing concepts?
- How can visual growth patterns support understanding of code structure?

### Mycelium and Networks

- What features of mycelial systems are most useful as teaching metaphors?
- How do mycelial networks communicate, adapt, and distribute resources?
- Which parts of the metaphor should be used carefully to avoid oversimplification?

### Human-Computer Interaction

- What interface patterns make learning systems feel welcoming?
- How can learners explore computing concepts through interaction instead of memorization?
- How can accessibility be built into the tool from the beginning?

### Prototype Development

- What is the smallest useful prototype this project could produce?
- Should the first prototype be a visualization, a game-like activity, or a coding tool?
- What would a successful user test look like?
